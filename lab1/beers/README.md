# Beer Generator

__To Build__

Run `make dep` if you haven't already.
Run `make make-beers`.

__Running__

`./makeBeers <num_movies>`

__Example Output__

```json
[
   {
      "id": 1,
      "score": 15,
      "brewery": {
            "name": "Firestone",
            "locations": [
               "San Luis Obispo",
               "Stockholm",
               "Copenhagen",
               "Orlando",
               "Barcelona",
               "Munich"
            ]
      },
      "type": "dark lager"
   }
]
```
